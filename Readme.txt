# Sales Prediction from Social Network Ads

This project predicts sales from Social Network Ads data and stores the results in a list.

## Running the Project

To run this project, follow these steps:

1. Install Python if you haven't already. You can download it [here](https://www.python.org/downloads/).
2. Download all project files to your local machine.
3. Open CMD or Terminal.
4. Navigate to the folder where you downloaded the project files.
5. Install necessary packages by running the following command:
6. then type "python app.py" or "py app.py" and hit enter
