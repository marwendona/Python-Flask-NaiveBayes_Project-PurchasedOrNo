# PurchasedOrNo

**PurchasedOrNo** is a web application developed using Python, Flask, and Naive Bayes algorithm to predict whether a user is likely to purchase a product based on certain features. The project leverages the Flask framework to create a robust web interface and employs Naive Bayes classification for accurate predictions.

## Running the Project

To run the project locally, follow these steps:

1. Ensure you have Python installed on your machine. If not, download it [here](https://www.python.org/downloads/).
2. Download all project files to your local machine.
3. Open CMD or Terminal.
4. Navigate to the folder where you downloaded the project files.
5. Install necessary packages
6. Run the application:

  ```
   python app.py
  ```

or

  ```
   py app.py
  ```

## Usage

After running the application, access it in your web browser at http://localhost:5000 (or the port specified).

## Technologies Used

- Python: The core programming language for the application.
